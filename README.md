# Liste_De_Noel
Description du projet  Projet : panneau d'administration web pour publier des articles et des mises à jour (démo : https://liste.louis-cercle.site). Objectif principal : apprentissage et démonstration d'un panel utilisable pour une entreprise afin de gérer le contenu du site une fois livré. Fonctionnalités clés  Interface d'administration pour créer/modifier/supprimer des articles et annonces. Gestion des utilisateurs et des rôles (administrateur, éditeur, etc.). Authentification sécurisée (hash des mots de passe, validation côté serveur). 2FA pour la connexion au panel (génération et vérification de codes temporaires, expiration, mécanismes de récupération). API REST pour consommation externe et intégration. Mesures de sécurité (requêtes préparées, validation des entrées, envoi sécurisé des sessions). Point particulier : 2FA  Mise en place pour renforcer la sécurité des accès au panel. Fonctionnement : l’utilisateur active la 2FA, reçoit/installe un mécanisme de codes temporaires (TOTP ou codes envoyés), les codes expirent après un court délai, possibilité de codes de secours pour récupération. Objectif : empêcher les prises de contrôle par compromission de mot de passe et améliorer la sécurité opérationnelle de l’entreprise. Remarques finales  Projet conçu principalement comme exercice pédagogique et preuve de concept pour un panel d’entreprise. Oui, le site est encore moche — c’est voulu pour l’instant : il fonctionne bien, on améliorera le style plus tard. 😄
